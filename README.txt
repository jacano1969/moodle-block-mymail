MY MAIL BLOCK PLUGIN
=================================

My mail is a Moodle 2.X block that shows your unread messages from local_mail plugin.

Web: https://github.com/IOC/moodle-block-mymail


Install
=======
Unpack archive inside /path/to/moodle/blocks/mycourses
